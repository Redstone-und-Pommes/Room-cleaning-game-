# 🧹 Room Cleaning Game

Ein interaktives Putz-Spiel, das deine Hand/Staubwedel erkennt und mit deinen Bewegungen arbeitet!

## 🎮 Wie man spielt

1. **Spiel starten** - Kamerazugriff erlauben
2. **Raum scannen** - Das Spiel erkennt deinen Raum
3. **Mit Hand putzen** - Bewege deine Hand über die weißen Schmutzpunkte
4. **Bubble Sounds** - Jedes gerinigte Partikel macht einen Bubble-Sound
5. **Fortschritt sehen** - Der Fortschrittsbalken zeigt wie viel du schon gereinigt hast

## ✨ Features

- 🎥 **Hand-Erkennung**: Nutzt KI um deine Hand zu erkennen (TensorFlow.js + MediaPipe)
- ⭐ **Weiße 3D-Partikel**: Schöne visuelle Effekte mit Glow
- 🎵 **Bubble Sounds**: Befriedigender Sound bei jedem gereinigten Punkt
- 📊 **Live Statistiken**: Score, Zeit, Fortschritt in Echtzeit
- 🎯 **2 Minuten Spielzeit**: Schnelles Gameplay oder 100% zum Gewinnen
- 📱 **Mobil optimiert**: Funktioniert auf Handy und Desktop

## 🚀 Tech Stack

- **HTML5 Canvas** - Grafiken und Partikel
- **TensorFlow.js** - Künstliche Intelligenz
- **MediaPipe Hands** - Hand-Pose-Erkennung
- **Web Audio API** - Sound-Effekte
- **GitHub Pages** - Kostenloses Hosting

## 🎯 Spielziele

- ✅ 100% des Raums reinigen
- 📈 Hohe Scores erreichen
- ⚡ Schnell spielen und gewinnen
